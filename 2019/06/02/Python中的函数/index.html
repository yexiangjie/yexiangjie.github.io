函数基础

学过编程的人都知道函数对一种语言来说的重要性不言而喻,函数一般是实现某一功能的一段代码,写好后即可重复使用,

下面是一个python函数的实例

  • def是函数声明
  • my_fun是函数名
  • 括号里面的message是参数
  • print语句是函数的主体,用于执行相应的语句
  • 函数的最后可以返回调用结果(return yield),也可以不返回

与编译型的语言(例如说C)不一样的是,def是可执行语句,只有在调用时才会创建新的对象,所以python不存在像C语言一样的函数声明,但是因为如果主函数直接调用了没有定义在后面的函数,程序会报错,故函数应该定义在主程序的前面,

如果我们在函数内部调用其他函数，函数间哪个声明在前、哪个在后就无所谓，因为def是可执行语句，函数调用前都不存在，我们只需保证调用时，所需的函数都已经声明定义

另外一点就是python的函数参数可以设置默认值,如果参数没有传入,就会使用默认值,否则默认值就会被覆盖

Python语言的一大特点就是dynamicall typed的(不像C语言中的函数必须声明传入的参数是什么类型的),Python中的函数可以接受任意类型

python不考虑输入的数据类型,而是交给具体的代码去实现,在编程语言中,我们可以称之这种行为为多态,这是python较其他编程语言很大的区别,但是这种方便的特性也会造成问题,因此必要时要在函数的开头加上数据类型的检查.

函数的嵌套

python的函数是支持嵌套,即可以在函数里面定义函数,C语言显然是不支持这种操作的

函数嵌套的作用主要有两个方面,一个就是保证内部函数的隐私,内部函数只能被外部函数所调用和访问,

例如我们想要保护数据库的用户名和密码

这里的函数 get_DB_configuration，便是内部函数,无法在connect_DB函数以外被单独调用

另外一个就是合理使用嵌套函数可以提高函数的效率

通过递归的方式求解一个数的阶乘,在开始之前先检查类型是否合法,这样就可以避免在递归过程中次次检查,从而提高程序运行的效率

函数变量作用域

与其他语言一样,变量在函数内部定义的则为局部变量,函数执行完就会释放回收

对应的全局变量则是定义在文件层次上的,整个程序都能访问,函数内部当然可以访问,但是需要注意的是不能直接在函数里面修改全局变量的值

程序报错是因为python的解释器默认函数里面的是局部变量,但是却发现局部变量没有申明,就无法执行相关操作,如果一定要在函数内部修改全局变量,我们要在全局变量的前面加上一个global的声明

另外，如果遇到函数内部局部变量和全局变量同名的情况，那么在函数内部,局部变量的值会覆盖掉全局变量的值

类似的，对于嵌套函数来说，内部函数可以访问外部函数定义的变量,如果要修改,则需要加上nonlocal关键字

闭包

类似于嵌套函数,只是闭包函数的返回值是内部的函数,且该函数通常赋值于一个变量

eg:计算一个数的n次幂

闭包是让代码更加简洁的一种方式,其常常与后面要学的装饰器一起使用

python的函数式编程

所谓的函数式编程是指代码中的每一块都是不可变的(immutable),都由纯函数(pure function)组成,

函数本身互不影响,对于相同输入总具有相同输出.

eg:将一个列表中的元素变为原来的两遍

这段代码就不是纯函数,因为列表中元素的值被改变了,多次调用会返回不同的值,如果要把它编程纯函数的形式,则需要建立新的列表并返回

函数式编程的优点在于其纯函数的不可变性使程序更加健壮,易于调试和测试,

缺点主要就是限制多并且比较难写

python不同于一些函数式编程语言(Scala,我也不了解),它并不是函数式编程语言,但也提供了一些函数式编程的特性

python主要提供下面三个函数:map(),filter(),reduce(),通常和匿名函数一起使用

map(function,iterable),主要是对iterable(可迭代对象)中的每个元素运用function函数,最后返回一个可遍历的集合
