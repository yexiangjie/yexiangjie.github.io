排序

发表于 2019-09-22 | 更新于: 2019-09-23 | | 阅读次数: |
字数统计: 1.7k | 阅读时长 ≈ 8

排序是PAT考试中很重要的一部分,不知道这次的考试是否会出现.排序题的难度一般是25分的中档题,但是题库中也是会出现30分的难题.

排序题很重要的就是读懂题目的意思,按照题目的要求排序,灵活使用sort函数也是排序题的基本要求,排序题还是很灵活的一种题型

排序题经常出现的问题就是如果分数一样的话输出相同的排名,但是后面的排名会自动两个,例如113,这时候排序的技巧就是利用sort函数排序后,先赋值排名为i+1,if i=i-1的分数,就令i的排名=i-1的排名即可

PAT1062

先读懂题目的意思,其实题目是按照一类一类人输出 ,其中德分和才分没过及格线的就直接不输出,所以在输入时即进行选择,然后就是一类一类的人.排序函数的话是德才总分高的先输出,如果相同的话输出德分较高者,如果还相同则输出号码较小者.可以定义几个向量,满足要求就入向量,之后一个个排序输出.

PAT1012

参考PAT1012,正确理解本道题目的意思就显得比较重要了.题目的大意是给出每门课的成绩,然后要我们取每门课最高的排名输出,如果排名相同的话按照一定的顺序输出,给出的查询可能会不存在,所以就要求我们开辟散列数组来存储已经采集到的成绩.

针对题目的意思,我们就先对每门课进行排名,之后再寻找每个学生的最高排名

PAT1025

参考PAT1025,是一道比较经典的排序题目,因为要输出考场的排名,所以对一个个考场收集之后排名,之后再综合进行排名,其中考场排名部分是需要好好分析的.

PAT1055

参考PAT1055,我直接直接写就发现报错并且运行超时,后来发现这道题目的最后一个检测点是必须要预处理过的才可以,因为极端情况下的数据是10的五次方,如果先排序,并且取出一个年龄的100个既可以降低程序的时间复杂度.
