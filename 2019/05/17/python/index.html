我对python的理解

关于python，我第一次听说应该是大一上的寒假，当时鼓吹python的人工智能和大数据，作为一个新兴的入口和行业吸引着无数的技术人去学习，这几年python的热度也是有增无减，它确实是一门入门简单的语言，但是它的进阶确实挺难的.python目前可以做爬虫啊，web开发，以及最热门的数据挖缺和机器学习

python的学习过程不是一蹴而就的，应该不断得学习和完善，我目前也仅限于对所有语法有一定的了解，并没有在某个具体的项目实践过，但是python会是一门很好的语言，写脚本你会发现很有趣，可以帮助你做一些繁琐的事情. 我关注的许多大佬就是通过python进行编写一些有趣的脚本.如果时间充裕并且有想法的话我可以搬运一些源码拿来应用.
