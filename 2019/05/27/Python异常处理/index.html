错误和异常

一般来说程序中的错误包括两种,一种是语法错误,另一种就是异常,语法错误一般是无法通过编译和运行的,而异常则是多指在程序运行过程中出现的错误,并抛出异常

当然,我们不知道的报错可能很多,所以一种通用的方法就是加入except Exception(是所有非系统异常的基类)

需要注意的是except block只会执行一次,这也就是为什么要把Exception放到最后,他会匹配到最先的except block并执行.

异常的使用场景和注意点

大型社交网站的后台，需要针对用户发送的请求返回相应记录。用户用户记录往往储存在 key-value 结构的数据库中,每次有请求过来后，我们拿到用户的 ID，并用 ID 查询数据库中此人的记录，就能返回相应的结果.

在 json.loads() 函数中，输入的字符串如果不符合其规范，那么便无法解码，就会抛出异常，因此加上异常处理就很必要

当然,我们切忌不可以滥用异常处理,例如判断键值是否在字典中,一般是采用if .. in ..:这样较为合理,否则容易造成代码冗长且不易阅读.
