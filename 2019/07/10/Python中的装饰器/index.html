身份验证

比如一些网站,如果不登录可以浏览一些文章,但是发表文章就必须登陆,这时我们就可以利用装饰器

日志记录

日志记录可以用来测试函数的执行时间

输入合理性检查

在机器学习框架中,调用机器学习集群进行模型训练的时候往往会对输入(一般是很长的json文件)进行合理性检查,从而避免资源浪费

缓存

缓存装饰器的在python中是存在于lru_cache中,会缓存进程中函数运行的参数和结果,当缓存满了之后会删除最久没有访问.

比方在设备检查中就会使用缓存解释器来包裹函数,避免反复调用,从而提高程序效率
